# Docker Image Template for Scientific Data Fitting

To use this image, include this line in your Dockerfile:

```
FROM ghcr.io/debsahu/pydatascience:main
```

or

Docker:

```
$ docker pull ghcr.io/debsahu/pydatascience:main
```

## Python Libraries

- numdifftools
- lmfit
- seaborn
- matplotlib
- scikit-learn
- pandas
- notebook
- numpy
